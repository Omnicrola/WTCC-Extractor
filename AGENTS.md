# Agents File
These directives are for AI agents working in this project. 

## General Directives
This project uses Python 3.13.5, and when writing python code all syntax and conventions should target this version.

## Version Control
This project uses git version control. You may use git commands to gather info about previous file versions and the state of the repository. You will never stage, commit, push, or pull from git yourself under any circumstances whatsoever.

## Output
The output of this project is in the \output\wtcc.db file. This file is very valuable, and is not maintained in version control. Take extreme care when modifying, manipulating, moving, or copying this file. You will never under any circumstances delete or directly modify this file. The python scripts you write for this project may modify this database file, but neither you or any agent will ever directly modify it.
